# FoodMart_Sales_Analysis_Excel
Interactive Excel dashboard analyzing FoodMart sales performance, profitability, customers, and market segments.

FoodMart Sales Analysis Dashboard (Excel)

Project Overview
This project presents an interactive Excel dashboard analyzing FoodMart sales performance, profitability, customer behavior, and market segmentation.

Key Metrics
- Total Sales
- Total Quantity Sold
- Total Profit
- Net Profit

Analysis Covered
- Sales by Gender
- Sales by Market Type
- Top 10 Customers by Sales
- Top 5 Products
- Top Stores by Sales & Quantity

Data Cleaning & Preparation
- Data cleaned and transformed using **Power Query**
- Converted text-based numeric fields and income ranges into numeric averages
- Handled missing values and formatting issues

Tools Used
- Microsoft Excel
- Power Query
- Pivot Tables & Pivot Charts
- Slicers
